Javascript Application Security

The Javascript ecosystem has grown dramatically with increases in both client and server side Javascript. This of course introduces new challenges.

For client applications, primary concerns may be around encoding all user input and preventing any kind of XSS. More details are in the client application page . Look here if you are writing Angular, JQuery, Backbone, Ember, etc..

Web and Server applications written in Javascript are increasing in large part due to node.js. For general background, see the General Javascript page. For framework specific guidance, look to the Node.js pages.
